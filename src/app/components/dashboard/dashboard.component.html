Bienvenido a tu Sistema de Gestión

Esta aplicación te permite gestionar notificaciones en tiempo real, archivos en S3 y colas SQS.

Notificaciones

Envía notificaciones en tiempo real a usuarios específicos, salas o broadcast a todos.

Archivos S3

Sube, gestiona y comparte archivos en AWS S3 con URLs firmadas seguras.

Colas SQS

Envía y recibe mensajes desde colas de AWS SQS para procesamiento asíncrono.

Correos SES

Envía correos electrónicos mediante AWS SES con soporte para plantillas y envío masivo.
  • WebSockets para notificaciones en tiempo real
  • Gestión completa de archivos en AWS S3
  • Sistema de colas con AWS SQS
  • Envío de correos con AWS SES
  • Plantillas de correo personalizables
  • Envío masivo de correos
  • URLs firmadas para compartir archivos seguros
  • Envío de mensajes individuales o por lotes
  • Interfaz moderna y responsive
